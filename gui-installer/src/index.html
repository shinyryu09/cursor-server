이 설치 프로그램은 MCP Cursor Server를 컴퓨터에 설치하고 설정하는 과정을 안내합니다.

주요 기능

  • MCP 프로토콜 지원
  • 다중 AI 모델 지원 (OpenAI, Anthropic, Google, Cursor)
  • Xcode 및 Android 프로젝트 자동 감지
  • Cursor Editor 연동
  • 실시간 채팅 및 코드 생성
  • 스마트 캐시 시스템

시스템 요구사항

  • Node.js 18.0.0 이상
  • macOS (Xcode 프로젝트 지원)
  • Cursor Editor (선택사항)
  • AI 모델 API 키 (최소 1개)
